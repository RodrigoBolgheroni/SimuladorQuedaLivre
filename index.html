Parâmetros

Dados do problema:
- Aceleração da gravidade (g): 9.8 m/s²
- Velocidade inicial (v₀): 0 m/s
- Altura inicial (h): 100 m

Equações do movimento:
- Posição: y = h - (v₀t + ½·g·t²)
- Velocidade: v = v₀ + g·t

Cálculo do tempo total de queda:
- Quando y = 0, temos: 0 = h - (v₀t + ½·g·t²)
- Reorganizando: ½·g·t² + v₀t - h = 0
- Usando a fórmula quadrática: t = (-(v₀) + √(v₀² + 2·g·h)) / g
- t = ((-0 + √(0² + 2·9.8·100)) / 9.8 = 4.52 segundos

Velocidade final:
- v = v₀ + g·t = v = 0 + 9.8 · 4.52 = v = 44.27 m/s

Conclusão:
A maçã levará 4.52 segundos para atingir o solo e chegará com velocidade de 44.27 m/s.
